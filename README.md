# 📊 Superstore Tableau Dashboard

## Turning Raw Data into Actionable Business Insights

### 🔍 What I Analyzed:
- Regional sales performance
- Profit distribution across customer segments
- Sales vs Profit relationships
- Monthly order quantity trends

### 📈 Key Insights:
- Central region contributes ~31% of total sales
- Corporate customers generate highest profit
- High sales ≠ High profit (loss transactions exist)
- Monthly orders peak in May

###  Tools Used:
- Tableau
- SQL
- Data Analysis & Visualization
