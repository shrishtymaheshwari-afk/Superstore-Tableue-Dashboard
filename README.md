# Superstore Tableau Dashboard

This project is an interactive business analytics dashboard built using the popular Superstore dataset in Tableau. The dashboard focuses on analyzing sales performance, profit trends, customer segments, and regional business insights to support data-driven decision-making.

The goal of this project was to transform raw business data into meaningful visual insights that can help identify profitable areas, operational inefficiencies, and customer behavior patterns.

## Project Objectives
Analyze regional sales and profit performance
Identify the most profitable customer segments
Understand the relationship between sales and profitability
Track monthly order quantity trends
Discover potential loss-making transactions

## Key Insights
-The Central region contributes the highest share of total sales
-Corporate customers generate the highest profit across regions
-High sales do not always guarantee high profit
-Some transactions show negative profit, indicating possible discount or cost-related issues
-Monthly order trends remain relatively stable with noticeable seasonal variation

## Dashboard Features
-Region-wise Sales Distribution
-Customer Segment Profit Analysis
-Sales vs Profit Scatter Plot
-Monthly Order Quantity Trend
-Interactive Filters for Better Analysis
